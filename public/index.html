Timestamp Microservice

This is a quick API built using node.js and express.

Pass a date (in natural language) or Unix timestamp after the url.

This will convert it either to a Unix timestamp or to a natural date.

Example Usage:

example.com/January 1, 2015
example.com/1483142400

The Unix Timestamp is in seconds, not milliseconds.


Example Output:

{"unix":1421625600,"natural":"January 19, 2015"}
